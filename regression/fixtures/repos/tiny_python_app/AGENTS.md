# Tiny Python App Instructions

This repository is a deterministic regression fixture for `OpenGiraffe`.

Rules:

1. Keep changes minimal and directly related to the task.
2. Preserve the existing simple module structure unless the task explicitly requires otherwise.
3. After code changes, run `python -m pytest -q` from the repository root.
4. A valid submission must leave the git working tree clean.
5. A valid submission must include the task changes in git commit history before you finish.
6. Once the task changes are committed and the tests pass, stop immediately. Do not continue exploring the repository or perform unrelated checks.
